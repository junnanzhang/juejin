# 逗号
#python
打印多个表达式时，用逗号将表达式隔开
```
>>> 1,2,3
(1,2,3)
>>> print 1,2,3
1 2 3
>>> print (1,2,3)
(1,2,3)
>>> 

```

多个赋值操作同时进行时，将多个值的序列解开，然后放到变量的序列中，这就叫序列解包或递归解包。

```
>>> x,y,z=1,2,3
>>> print x,y,z
1 2 3
>>> x,y=y,x
>>> print x,y,z
2 1 3
>>> values=1,2,3
>>> values
(1,2,3)
>>> x,y,z=values
>>> x
1
>>> y
2
>>> z
3
```
